# Kaleidoscope

An interactive mathematical kaleidoscope — a visual tool 
for exploring symmetry, pattern, and mathematical relationships 
through generative geometry.

Built in vanilla HTML5. No dependencies. Works in any browser.

## What It Does
- Generates kaleidoscopic patterns from mathematical symmetry groups
- Interactive — your input shapes the pattern in real time
- Demonstrates rotational symmetry, reflection, and tessellation

## Why
Mathematics lives in pattern and symmetry. A kaleidoscope 
makes those relationships visible, beautiful, and intuitive 
before a single equation is written.
